# Python clovers 🍀

Learning Python from scratch by drawing ASCII clovers, to develop an understanding and display learned skills. Brick by Brick, a folder per lesson.

## Lessons
- **lesson-01-print** — the `print()` command
- **lesson-02-multiline-duck** — multi-line text with triple quotes
- **lesson-03-functions** — defining and calling functions
- **lesson-04-parameters** — passing inputs into a function
- **lesson-05-loops** — lists and `for` loops
- **lesson-06-decisions** — choosing ducks with `if` / `else`
